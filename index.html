HawaHawa Audio System

A lightweight and powerful audio framework designed for seamless integration into Unity projects. It provides a robust audio manager that simplifies sound playback, randomization, and mixer implementation.

Key Features

  • Lightweight & Optimized – Minimal performance overhead for any Unity project.
  • Easy-to-Use API – Simple implementation with minimal coding required.
  • Audio Groups – Organize and play random sound variations dynamically.
  • Random Pitch & Fading – Natural sound variation and smooth transitions.
  • Mixer Group Integration – Route sounds through Unity's Audio Mixer.
